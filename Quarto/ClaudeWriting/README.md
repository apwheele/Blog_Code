# Using Claude Code to help me write

This is an illustration of my set up to use Claude Code to help me write. In a nutshell it is "review prior writing" and then "use an outline to write something new".

The folders contain examples of prior writing, and then a demo of writing a blog post from scratch in the `/ClaudeWritingPost` folder. Can see the blog post it wrote `ClaudeWritingExample.md` (also check out the Claude Code session at `Claudewriting_tips.txt` for how it was generated). To compare to what I wrote manually, see [*Using Claude Code to help me write*]() on my personal website.
