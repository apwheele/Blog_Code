# Resume/CLAUDE.md

## Project Overview
This folder contains my professional resume and CV materials. These documents serve as reference for other projects when biographical information, professional background, or expertise areas are needed.

## Document Purpose
The resume is:
- A living document updated as experience grows
- Source of truth for professional history
- Reference for proposals, bios, and professional content
- Not for public blog posts (use judgment on what to share)

## Usage in Other Projects
When working in BookWriting, CrimeDeCoderBlog, LinkedInPosts, PersonalTechBlog, or Projects folders:
- Reference resume for accurate dates, titles, accomplishments
- Pull relevant experience for proposals
- Use for author bios and about sections
- Verify technical skills and tool proficiencies
- Cite specific projects or achievements when appropriate

## Writing Style (for resume updates)
- **Concise and active**: Start bullets with action verbs
- **Quantified results**: Include numbers, percentages, scales
- **Achievement-focused**: Emphasize impact, not just duties
- **Keyword-rich**: Include relevant technical terms for ATS
- **Consistent formatting**: Parallel structure in bullet points

## Resume Structure
- Contact information
- Professional summary or objective
- Core competencies/technical skills
- Professional experience (reverse chronological)
- Education and certifications
- Publications, speaking, or notable achievements
- Optional: volunteer work, professional memberships

## Key Guidelines
- Tailor for different opportunities (consulting vs. technical role)
- Keep to 1-2 pages for resume, longer for academic CV
- Update regularly with new accomplishments
- Remove outdated or less relevant items
- Proofread obsessively (zero tolerance for errors)
- Maintain consistent date formatting
- Use PDF format for sharing

## Privacy Note
Resume contains personal information. When referencing in other projects:
- Use judgment about what details to include publicly
- Anonymize client names if used in examples
- Keep contact information private in public content
- Full details appropriate for proposals and professional documents
