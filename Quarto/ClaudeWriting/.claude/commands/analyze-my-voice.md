# Analyze My Writing Voice

Content type to analyze: $ARGUMENTS (or "all" for comprehensive analysis)

## Task
Read examples from the specified folder(s) and provide a detailed analysis of:

1. **Sentence Structure**
   - Average sentence length
   - Complexity (simple vs compound)
   - Rhythm and flow patterns

2. **Paragraph Style**
   - Typical paragraph length
   - How I break up text
   - Use of white space

3. **Voice Characteristics**
   - Formality level (1-10 scale)
   - Use of first person vs other POV
   - Personal vs objective tone
   - Humor or personality markers

4. **Technical Communication**
   - How I explain complex ideas
   - Use of analogies/examples
   - Code presentation style (if applicable)

5. **Structural Patterns**
   - How I open pieces
   - How I close pieces
   - Use of headers, lists, formatting

6. **Unique Markers**
   - Phrases or patterns I use frequently
   - Things I notably avoid
   - My distinctive style elements

Provide specific examples from my writing to support each observation.

This analysis will help you write content that sounds like me.
